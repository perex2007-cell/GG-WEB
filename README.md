# ConstructPro Landing Page

Proyecto React + Vite convertido desde un HTML estático.

## Estructura

- `index.html` — plantilla principal de Vite.
- `src/` — componentes React y estilos.
- `package.json` — dependencias y scripts.
- `tailwind.config.js`, `postcss.config.js` — configuración de Tailwind.

## Comandos

- `npm install` — instala dependencias y crea `node_modules`.
- `npm run dev` — ejecuta la app en modo desarrollo.
- `npm run build` — genera la carpeta `dist`.

## Notas

- `public/` se usa para archivos estáticos si los necesitas.
- Las fuentes de Google y Material Symbols se cargan desde `index.html`.
